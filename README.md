# PlateColor

> **Lightweight World of Warcraft nameplate enhancement for clean visuals and fast combat awareness.**

---



**PlateColor** is built for players who want a clean interface without complex configuration. Its design focuses on **simplicity**, **low overhead**, and **practical improvements** that make nameplates easier to read in real combat.

### ✨ Highlights
* **Targeted Customization:** Focuses on core elements: name text, health bars, cast bars, auras, threat, and execute lines.
* **Pure Performance:** Keeps its scope narrow to ensure zero impact on your framerate.
* **Out-of-the-Box:** Avoids unnecessary menus; it just works.

### 🌟 Features
* **Minimalist Design** – Optimized for high-intensity combat performance.
* **Readable Visuals** – Enhanced clarity for nameplates and cast bars.
* **Core Support** – Full support for auras, threat indicators, and markers.
* **Personal Resource Display** – Seamlessly integrated support.
* **Zero Bloat** – No complex configuration or heavy UI layers.

### 🛠️ About the Author
Built by a long-time **Mythic+ player**. Features are added based on practical needs and real-world dungeon requirements.

**PlateColor** 是一款专为《魔兽世界》开发的轻量化姓名板插件。它适合追求清晰视觉体验、希望快速获取战斗信息，却又不想被复杂设置困扰的玩家。

> **设计初衷：极简、低负担、以及真正的实用增强。**

### ✨ 核心亮点
* **针对性定制：** 专注于核心元素（姓名、血条、施法条、光环、仇恨、斩杀线等）的视觉优化。
* **极致性能：** 严格控制功能范围，确保在任何战斗环境下都有极低的性能开销。
* **开箱即用：** 拒绝繁琐的配置菜单，保留最直观、最有价值的战斗信息。

### 💎 插件特性
* **极简轻量化设计** – 极致代码优化，不占用过多系统资源。
* **清晰直观增强** – 强化视觉层级，让目标识别更加迅速。
* **核心功能支持** – 完善支持血、法、光环、仇恨及各类标记。
* **个人资源条** – 支持个人资源条 (PRD) 增强展示。
* **零配置负担** – 刻意避免冗余菜单，专注于战斗本身。

### 👤 关于作者
由一位**资深大秘境玩家**制作。所有功能均源自实战需求，并根据版本环境持续进行针对性的添加与优化。
